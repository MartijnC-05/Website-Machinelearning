De toekomst van Machine learning


Machine learning wordt steeds belangrijker in ons leven omdat het veel taken die veel denk of rekenwerk van ons vragen kan overnemen, de verwachting is daarom ook dat machine learning in de toekomst veel meer mogelijk maakt dan nu al kan met computers. Machine learning zal veel mogelijk maken en zal ook steeds meer zelf kunnen doen. Machine learning zal ook op steeds meer plekken worden toegevoegd. Maar ook voor ons zal het grote verandering in ons leven geven, zo zal er steeds meer informatie over onszelf verzameld worden waardoor de computer precies weet wat je leuk vind en wat je graag wilt hebben en dat daardoor allemaal reclames en informatie die wij lezen daarop worden afgestemd. Maar machine learning kan ook minder goed uitpakken want het kan ook de banen overnemen van sommige mensen waardoor er een tekort aan werk kan komen en dat is weer niet goed voor de economie. Of misschien worden de computers wel zo slim dat ze zelf een eigen leven gaan leiden en dat ze de wereld over nemen. Maar zover zal het vast niet komen. Machine learning zal dus veel invloed gaan krijgen op ons leven in de toekomst en ook veel grote tech bedrijven zoals Google, Amazon en Microsoft zullen veel meer van dit soort technieken gebruik gaan maken om nog meer invloed te krijgen.
