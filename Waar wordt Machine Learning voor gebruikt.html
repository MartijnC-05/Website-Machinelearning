Waar wordt machine learning voor gebruikt?

In de zorg

Machine Learning kan een gigantische hulp zijn voor de zorg. Zo kan machine learning veel data verzamelen, artsen helpen bij het maken van voorspellingen en ervoor zorgen dat artsen minder belast raken. Een uitleg over deze drie voorbeelden:

Data verzamelen: Artsen hebben hedendaags heel veel data tot hun beschikking. Maar een groot deel hiervan is onbruikbaar voor een normaal mens, aangezien het ongestructureerd is. Machine Learning kan hier wel wat mee, aangezien dit hier wel wat bruikbaars van kan maken. Al helemaal met de huidige staat van de wereld komt dit goed van pas, aangezien artsen nu heel veel data moeten opslaan van alle patiënten die komen en gaan.

Voorspellingen maken: Machine Learning kan beter voorspelling doen dan normale mensen. Machine Learning kan misschien nog niet een dokter volledig vervangen, maar het kan ontzettend erg helpen bij het maken van voorspellingen. Zo kan machine learning ervoor zorgen dat een keuze sneller wordt gemaakt, maar het kan ook een nieuw advies aanraden als het vorige plan niet goed blijkt te werken.

Minder belasten: Machine Learning kan ervoor zorgen dat artsen minder werk te hoeven doen. De computer kan, in samenwerking met de kennis van een arts, kan een flinke hulp zijn bij behandelingen. Hierdoor kan de arts minder werk hoeven te doen, waardoor deze sneller naar een andere patiënt kan. Zo kunnen ook gelimiteerde middelen sneller worden verplaatst van patiënt naar patiënt, en dus zal de zorg verbeteren, waardoor meer mensen gezonder het ziekenhuis kunnen verlaten.



Op social media

Ook voor social media kan Machine Learning een goede hulp zijn. Hier zijn vier voorbeelden ervan: Zodat de netwerk giganten hun producten beter op doelgroepen kunnen richten, het beschermen van reputatie en data, om de visualiteit te verbeteren en ten slotte het automatiseren van data. Hieronder verdere uitleg van de voorbeelden.

Betere doelgerichtheid: Deze is misschien vanzelfsprekend, maar machine learning kan beter advertenties matchen met bepaalde gebruikers. Door data te verzamelen van wat welke gebruiker doet en leuk vind, kan bijvoorbeeld Facebook of Instagram advertenties tonen die passen bij de activiteiten van die gebruiker.

Beschermen van reputatie en data: Zelfs de grote websites met de beste beveiliging kunnen het slachtoffer van spam en andere malware worden. Hiertegen kan machine learning worden ingezet, waardoor de spam-posts en gebruikers die dit posten uit het algoritme worden geplukt en dus ook meteen op hun donder krijgen.

Verbeterde visualisatie: Het verbeteren van visualisatie houdt in dat machine learning de socials helpt bij het verbeteren van het uiterlijk van de app. Zo heeft Twitter een bedrijf gekocht dat hen gaat helpen bij het beter scherpstellen van video's en foto’s, en ook zal het helpen bij het verbeteren van schermverhoudingen.

Automatiseren van data: Er zijn heel veel gebruikers van social media, te veel voor een groep mensen om goed te organiseren en mee te werken. Machine Learning kan een groot deel van deze gebruikers automatiseren, en dus ook meteen automatiseren wat deze gebruikers te zien krijgen aan advertenties en aangeraden posts.



Fraudebestrijding

Voor fraudebestrijding kan machine learning ook een goede hulp zijn. Al helemaal met de recente groei van het aantal financiële fraudegevallen komt dit goed van pas, aangezien fraudebestrijding een geval is waar de computer echt beter kan werken dan de mens. Hier volgen de redenen waarom:

Accurate data-analyse: Door de vaardigheid van machine learning om veel data te kunnen processen, kan het veel transacties tegelijk bekijken en daar ook goed verdachte transacties er tussenuit halen. Hierdoor kunnen ze dus beter fraude vinden, wat machine learning efficiënter maakt dan normale analisten.

Meer ademruimte voor fraude analisten: De normale analist heeft simpelweg te veel werk te doen met alle transacties, al helemaal met de grote opmars van online transacties. Hier komt machine learning weer om de hoek kijken, niet alleen kan het meer transacties behandelen dan de normale analist, maar het kan er dus voor zorgen dat de analisten die nog moeten werken, minder hoeven te doen, wat er indirect voor zorgt dat deze analisten hun werk beter doen.

Minder false-positives: Voor dit punt is het belangrijk te weten wat een false-positive is. Het is een geval waar een transactie wordt bestempeld als fraude, terwijl het dat niet is. Machine learning kan het aantal hiervan flink verminderen, simpelweg wegens het feit dat het beter kan analyseren dan een normale analist. Dit scheelt tijd, moeite, geld en vooral gefrustreerde gebruikers.



Reclame

Voor reclame kan er beter een specifiek onderdeel van machine learning worden gebruikt, namelijk deep learning. Deep Learning kan ervoor zorgen dat consumenten een meer persoonlijke ervaring krijgen, iets waar zij al een tijdje naar verlangen. Maar stel dat het gedrag van een consument plots veranderd, kan deep learning hierop inspelen. Zo kan deep learning bij een gebruiker die iets lijkt te willen kopen de laatste duw geven om dit ook te laten gebeuren, normale machine learning kan dit minder goed. Zo heeft iedereen waarschijnlijk wel eens gehad dat je op internet iets opzoekt en dat je meteen daarna al reclames krijgt over dat product, dit komt dan door deep learning.



Zelfrijdende auto's

In auto’s kan, net zoals alle bovenstaande genoemde, ook machine learning worden toegepast. Bij zelfrijdende auto’s ligt eigenlijk het meeste potentieel, maar dit is waarschijnlijk ook omdat zelfrijdende auto’s het minste zijn toegepast in de samenleving. Zelfrijdende auto’s hebben het meest baat bij sensoren, en deze worden constant verbeterd. De huidige systemen kunnen niet volledig zelf een auto besturen, maar ze zijn een grote ondersteuning van de menselijke besluitvorming en verminderen toch menselijke fouten. Al kan men verschillende zelfrijdende auto’s met elkaar verbinden, kan dat zorgen voor een vloeiende doorstroming van het verkeer en een verbetering van veiligheid en efficiëntie. Toch is het niet allemaal goed nieuws, machine learning kan toch tegen een paar problemen aanlopen. Zo is rijden op de snelweg misschien niet een grote uitdaging, maar het navigeren van de Haagse binnenstad kan toch een probleem vormen voor de AI. Ook heeft de AI data nodig om een situatie te kunnen voorspellen en daarin goed te handelen. Als er plots iemand vlak voor de AI op de rem trapt, en de AI kent dit verschijnsel nog niet, zal dit een botsing veroorzaken. Voordat we de AI in de auto’s kunnen inzetten, moeten veiligheid en betrouwbaarheid worden gegarandeerd, maar de kans dat dit gebeurt wordt met de dag groter.



Games

Machine learning kan een grote hulp zijn voor de gamesindustrie, alleen niet echt in de huidige vorm. “Video game AI zit nog in de donkere tijden.” Aldus Epic Games CEO Tim Sweeney. Zodra de AI echter betrouwbaar gebruikt kan worden, zal het waarschijnlijk voor een flinke verbetering in de industrie zorgen. Er zijn vijf manieren waarop machine learning de gamesindustrie kan helpen. Dit zijn:

Het overnemen van NPC’s: Het nadeel van NPC’s (Non Playable Characters) in huidige games is dat het programmeren van hun AI allemaal met de hand moet gebeuren, niet alleen kost dit tijd, maar ook kan het na een tijdje voorspelbaar worden wat de NPC van plan is. Machine learning kan dit verbeteren. Door machine learning kan de AI van NPC’s zo 1-2-3 worden veranderd. Hierdoor kunnen NPC’s reageren op acties van de speler (zoals wegrennen wanneer de speler boos gaat doen) of, al is de NPC een tegenstander, van moeilijkheidsgraad veranderen als de speler het goed of slecht doet. NPC’s kunnen op deze manier ook trucjes van spelers oppikken, en die tegen dezelfde spelers gebruiken. En dan misschien wel het belangrijkste punt, als game developers overstappen naar machine learning voor de NPC’s, zal dit de tijd die nodig is voor het programmeren van hun AI drastisch verlagen.

Het modelleren van complexe systemen: Video game developers zijn constant bezig met het verbeteren van de realiteit en immersiviteit van games, alleen zitten ze nu al tegen de top van hun potentieel aan. Hier komt machine learning weer om de hoek kijken. Het namaken van de echte wereld kan een flinke uitdaging zijn, maar machine learning kan dit al beter. Wat machine learning vooral kan is het creëren van dingen die de developers niet kunnen. Zo kan machine learning de effecten van de spelers keuze op de wereld laten zien, en kan het ook dingen die buiten de controle van de speler liggen, zoals het weer, beïnvloeden.

Het verbeteren van game uiterlijk: Nog een onderdeel van het verbeteren van games ligt in het uiterlijk van de games zelf. Meestal lijkt iets in game van ver weg heel mooi, maar als je dichterbij komt blijkt het toch niet zo mooi te zijn. het tegenovergestelde kan ook waar zijn, dat iets van dichtbij er goed uitziet maar van ver weg niet meer is dan een hoop pixels. Met machine learning kunnen developers ervoor zorgen dat het zien van dingen in games net zo werkt als in het echte leven: Van ver weg zijn de details op een voorwerp niet te zien, maar des te dichterbij je komt, des te beter de details naar voren komen. Dit noemen we dynamisch renderen.

Realistischere interacties: Weer terugkomend op het gedrag van NPC’s, maar dit keer over de manier hoe jij als speler met vriendelijke NPC’s praat. Dit is meestal vrij gelimiteerd. De developers moeten gesprekken programmeren, en meestal zijn deze gesprekken gelimiteerd door een gelimiteerde keuze aan uitspraken. Door AI kan dit worden verbeterd: Al nemen we als voorbeeld Google assistent, zie je dat jij als speler voor bijna elk bericht tegen de NPC (In dit geval dus Google assistent) een ander antwoord krijgt. Al kun je als speler berichten inspreken, kan je dus verschillende antwoorden krijgen. Al helemaal bij games in VR kan dit goed worden toegepast, want dan heb je niet alleen maar de spraak, maar ook lichaamstaal waar de NPC op kan reageren. Op deze manier kan de NPC nog beter de intenties van de speler zien, en daar dus ook weer toepasselijk op reageren.

Het creëren van werelden tijdens het spelen: Een van de dingen die een game populair kunnen maken, is de wereld waarin je speelt. Sommige van de populairste games hebben gigantische werelden, maar deze werelden moeten compleet met de hand worden ontworpen en gerenderd. Dit kost aardig wat tijd, maar dan komt machine learning weer in beeld. Het kan, terwijl de game gespeeld wordt, een wereld creëren. Zo kan je als speler niet voorspellen wat je als volgende tegenkomt, iets dat games waarschijnlijk alleen maar zal verbeteren.
