Wat is Machine Learning?


Machine learning, ook wel machinaal leren genoemd, is een onderdeel van kunstmatige intelligentie. Machine learning houdt zich bezig met het ontwikkelen van algoritmes en technieken waarmee computers zelf kunnen leren. Machine learning houdt in dat computers ontdekken hoe ze taken kunnen uitvoeren zonder hiervoor te zijn geprogrammeerd. Hierbij leren computers van de verstrekte gegevens om bepaalde taken uit te voeren. Het doel is om de computers te laten leren zonder dat er mensen bij aan te pas hoeven te komen. Hoe meer de computer oefent hoe accurater de uitkomst wordt. Machine learning kan een grote hulp zijn omdat het veel denk en rekenwerk van mensen kan overnemen.

Machine learning is een onderdeel van Artificial intelligence (kunstmatige intelligentie). Kunstmatige intelligentie is functionele intelligentie buiten het brein van de mens om. AI gaat over software die zelfstandig problemen kan oplossen. Denk bijvoorbeeld aan robots, er zijn robots die zelf kunnen leren hoe ze een taak moeten uitvoeren zonder dat de robot daar precies voor geprogrammeerd is. Zo zijn er robots die dan niet elke keer moeten worden geprogrammeerd om een klein parcour af te leggen, maar met behulp van machine learning kunnen die robots zelf bedenken hoe zij die route moeten afleggen.

En machine learning bestaat weer uit Deep learning. Deep learning is gebaseerd op meerlaagse neurale netwerken. Een aantal voorbeelden van deep learning: beeldherkenning (faceID), spraakherkenning (Alexa, Siri) en vertaling (Google Translate)

Machine learning is dus een onderdeel van kunstmatige intelligentie en Deep learning is weer een onderdeel van machine learning, in deze afbeelding kun je duidelijk zien hoe het in elkaar zit.
